Expand description

A trait for accessing exports (like Exportable) but it takes generic Args and Rets parameters so that NativeFunc can be accessed directly as well.

Required methods

[src]

fn get_self_from_extern_with_generics(
    _extern: &'a Extern
) -> Result<Self, ExportError>

Get an export with the given generics.

[src]

fn into_weak_instance_ref(&mut self)

Convert the extern internally to hold a weak reference to the InstanceRef. This is useful for preventing cycles, for example for data stored in a type implementing WasmerEnv.
